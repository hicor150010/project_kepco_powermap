# 프로젝트 요구사항

## 프로젝트명
썬캐쉬하우스 - 배전선로 여유용량 시각화 지도

## 의뢰자
K1567361460

## 개요
KEPCO 배전선로 여유용량 엑셀 데이터를 업로드하면, 지도 위에 각 지번별 여유용량 상태를 색상 마커로 시각화하는 웹 서비스.

## 데이터 원본
- KEPCO 크롤러 출력 엑셀 (xlsx)
- 헤더 구조: 2행 그룹헤더 + 3행 컬럼헤더, 4행부터 데이터
- 주소: 시/도, 시, 구/군, 동/면, 리, 상세번지 (6개 컬럼 결합)

## 색상 표시 로직 (우선순위)

| 색상 | 조건 |
|------|------|
| 빨간색 (Red) | 변전소여유용량 = "여유용량 없음" (최우선) |
| 파란색 (Blue) | 변전소 + 주변압기 + 배전선로 모두 "여유용량 있음" |
| 초록색 (Green) | 변전소 + 주변압기 "있음", 배전선로만 "없음" |
| 노란색 (Yellow) | 변전소 "있음", 주변압기 + 배전선로 "없음" |

## 기능 요구사항

### 필수 (MVP)
1. 엑셀 파일 업로드 (드래그앤드롭 지원)
2. 주소 → 위경도 자동 변환 (카카오 지오코딩)
3. 카카오맵에 4색 마커 표시
4. 마커 클릭 시 상세 정보 팝업
   - 변전소명, 배전선로명, 주변압기
   - 각 항목별 kW 수치 (접속기준용량, 접수기준접속용량, 접속계획반영접속용량)
   - 여유용량 상태
5. 마커 클러스터링 (대량 데이터 성능 최적화)

### 편의
6. 주소/지역 검색 → 해당 위치 이동
7. 색상별 필터 (빨간색만 보기 등)
8. 지오코딩 결과 캐싱 (동일 주소 재변환 방지)

## 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js (App Router) | Vercel 최적화 |
| 배포 | Vercel (서버리스) | 무료, 간편 |
| 지도 | 카카오맵 JavaScript SDK | 한국 주소 최적, 무료 |
| 지오코딩 | 카카오 REST API | 30만건/일 무료 |
| 엑셀 파싱 | SheetJS (xlsx) | 클라이언트 사이드 |

## 비용
- 카카오맵 지도 표시: 무료 (무제한)
- 카카오 지오코딩: 무료 (30만건/일, 초과 시 차단만)
- Vercel 호스팅: 무료 플랜
- **총 비용: 0원**

## 당부사항 (의뢰자)
- 주소가 여러 칸으로 나뉘어 있으니 합쳐서 지도가 잘 찾도록
- 색깔만이 아닌 클릭 시 kW 수치도 표시
- 지오코딩 결과 캐싱하여 비용/시간 절약
