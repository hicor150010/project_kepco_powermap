# KEPCO 배전선로 여유용량 — 통합 개발 계획서

> 최종 수정: 2026-04-12

---

## 1. 시스템 전체 구조

```
[GitHub Actions (3 스레드)]              [웹 서비스]
KEPCO API ──→ Supabase DB 직접 UPSERT ──→ 지도 표시
  (Python, 무료)     (엑셀 중간 단계 제거)    (Next.js/Vercel)

[데스크탑 크롤러] ← 별도 유지 (의뢰자용 독립 도구)
```

---

## 2. 프로젝트 구성

### 2.1 웹 서비스 (web/)
- Next.js + React + TypeScript + Supabase + Tailwind CSS
- Vercel 호스팅 (무료)
- 카카오맵 SDK + 카카오/VWorld 지오코딩

### 2.2 자동 크롤러 (crawler/)
- GitHub Actions (Public repo, 무료 무제한)
- 3개 독립 스레드 동시 실행
- 상세: [CRAWLING.md](./CRAWLING.md)

### 2.3 데스크탑 크롤러 (별도 리포)
- Python + Tkinter, PyInstaller exe 배포
- 의뢰자 직접 사용 독립 도구
- 위치: `project_crawling/kepco여유선로용량/src/`
