썬캐쉬하우스 — KEPCO 배전선로 여유용량 지도 개발 제안서
─────────────────────────────────────────────────────

한국전력에서 분기마다 공개하는 배전선로 여유용량 데이터를
지도에 올려, 태양광 부지 검토를 빠르게 할 수 있는 웹 서비스입니다.

엑셀 수백만 건을 일일이 뒤지는 대신, 지도 위에서 바로
"여기 들어갈 수 있나, 얼마나 들어갈 수 있나"를 확인합니다.



1. 엑셀 업로드
─────────────

  한전 표준 양식 엑셀을 그대로 올립니다.
  여러 파일을 한꺼번에 올려도 되고, 중복 데이터는 자동으로
  최신 값으로 갱신됩니다. 잘못된 행은 건너뛰고, 처리 결과를
  진행률로 보여줍니다.



2. 지도 마커
─────────────

  데이터를 마을(리·동) 단위로 묶어 지도에 표시합니다.

  마커 하나에 변전소·주변압기·배전선로 세 가지 상태를
  가로 3줄 막대로 동시에 보여줍니다. 빨간 길이가 부족 비율,
  파란 길이가 여유 비율입니다. 한 마커만 봐도 시설별로
  얼마나 부족한지 바로 판단할 수 있습니다.

  화면을 확대하면 마을 이름과 잔여 용량(kW/MW)이 함께 나오고,
  마커가 많은 곳은 자동으로 묶음(클러스터) 처리됩니다.



3. 여유 있는 곳 찾기
────────────────────

  사이드바 상단의 "여유 있는 곳만 보기" 버튼 한 번으로
  세 시설 모두 여유 있는 마을만 표시할 수 있습니다.

  좀 더 세밀한 필터도 제공합니다.
    - 시설별 4단계 (전체 / 전부 여유 / 일부 부족 / 전부 부족)
    - 지역별 단계 선택 (시도, 시군구, 동면, 리)
    - 변전소·배전선로 이름 검색

  잔여 용량 큰 순으로 정렬한 "유망 부지 TOP 10" 패널도
  지도 위에서 띄워 볼 수 있습니다.



4. 주소·지번 검색
─────────────────

  화면 하단 검색바에 주소나 지번을 자유롭게 입력합니다.
  결과는 두 가지 단위로 보여줍니다.
    - 리 단위 : 매칭되는 마을 그룹
    - 지번 단위 : 그 번지의 시설 정보

  찾는 번지가 데이터에 없으면, 같은 마을에서 가장 가까운
  번지를 자동으로 보여줍니다. 같은 마을은 같은 변전소·선로에
  연결되어 있어 옆 번지 데이터로도 충분히 추정할 수 있습니다.

  예: "100번지" 검색 → 데이터에 없음 → "105번지" 자동 표시



5. 거리재기
───────────

  네이버 지도와 동일한 사용감입니다. 좌클릭으로 점을 추가하고
  우클릭으로 종료합니다. 마을 마커를 클릭해서 점으로 잡을 수도
  있어, 변전소까지 거리를 잴 때 유용합니다.



6. 시설 상세 보기
─────────────────

  마을 마커를 클릭하면 그 마을의 여유/부족 비율이 카드로
  나타납니다. "상세 목록 보기"를 누르면 마을의 모든 지번·시설
  데이터를 표로 볼 수 있고, 컬럼별 정렬과 검색·페이지 처리,
  행 펼침까지 지원합니다.



7. 사용자 계정 관리
──────────────────

  의뢰자가 직접 사용자 계정을 발급하고 권한을 관리합니다.
  권한은 두 단계 — 관리자(업로드/계정 관리)와 일반 사용자(조회)
  로 나뉩니다. 회원가입 기능은 두지 않아 외부 침입 가능성을
  차단했습니다.



개발 기간
─────────

  최대 2주



개발 비용
─────────

  150만원 (현금가 기준)

  월간 운영비(서버·DB·도메인 등)는 1만원 이내로 예상되며,
  이 부분은 개발비에서 충당해 드리는 것으로 하겠습니다.
  의뢰자께서는 운영비를 별도로 부담하지 않으셔도 됩니다.

  다만 사용량이 예상 범위를 크게 벗어나(예: 사용자 급증)
  운영비가 정상 수준을 넘는 경우에는, 사전에 안내드린 뒤
  별도로 협의 드릴 수 있는 점 양해 부탁드립니다.
