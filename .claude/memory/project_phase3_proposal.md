## 🔌 데이터 소스별 실제 연계 스펙

### 1. 전국태양광발전소전기사업허가정보 (요청 1 핵심)

- **Endpoint**: `http://api.data.go.kr/openapi/tn_pubr_public_solar_gen_flct_api`
- **공공데이터포털**: data.go.kr/data/15107742
- **승인**: 자동승인, 무료, 일 1만건 (기존 건축물대장 계정 재사용 가능)
- **필드** (실제 표준):
  ```
  시설명, 지번주소, 도로명주소, 위도, 경도,
  설치상세위치구분명 (지붕/부지/건물옥상),
  설비용량(kW), 가동상태, 설치연도, 설치면적,
  허가일자, 허가기관
  ```
- **매칭 방식**: 같은 지번 OR 좌표 50m 반경 → "☀️ 설치완료" 표시
- **한계**:
  - 3kW 이하 자가발전 누락 (허가 대상 아님) — 축사/온실은 대부분 30kW↑라 영향 적음
  - 허가만 받고 미시공 케이스 → `가동상태` 필드로 필터링

### 2. 국토부 도로(현황) — 이격거리 도로 판정용

- **공공데이터포털**: data.go.kr/data/15125057
- **형태**: SHP 파일 월간 다운로드 (전국 12,300건)
- **무료, 자동승인 여부 불명** (추후 확인)
- **차로수 필드** 존재 여부 미검증 → **작업 전 SHP 다운로드해서 속성 확인 필수**
- **대안**: 도로중심선 (data.go.kr/data/15146878) 에 차선수 포함
- **필드 없으면**: 도로등급(국도/지방도/시도) 로 근사 (정확도 80%)
- **DB 저장**: Supabase PostGIS 로 import → 공간 쿼리

### 3. 법제처 자치법규 조회 API — 조례 수집

- **Endpoint**: `http://open.law.go.kr/LSO/openApi/`
- **공공데이터포털**: data.go.kr/data/15058294
- **포맷**: XML, 조문별 전문 제공
- **자동승인, 무료, 실시간 업데이트**
- **받아올 수 있는 것**:
  - 조례 제목, 조문 전문, 시행일, 개정이력
  - 예: "여수시 도시계획 조례 제18조의2 (태양광 발전시설의 입지기준)"
- **수치 추출 난이도**: 조례마다 표현 다름 ("200m", "이백미터", "도로경계부터 200m")
  → LLM 파싱 85~90% 성공 + 20~30% 수작업 보정 필요

### 4. 건축물대장 API — 주택 군집 판별 (이미 보유)

- 이미 2차 개발로 키·검증 완료 → `reference_bldg_register_api.md` 참조
- `mainPurpsCdNm` = "단독주택" / "공동주택" 필터
- DBSCAN (eps=50m, minPoints=5) 으로 "5호 이상 밀집 군락" 판정

## 🏗️ 구현 알고리즘 (확정)

### 태양광 설치여부 매칭
```
[축사/온실 리스트] ← 건축물대장 (2차)
        ∩
[태양광 허가 리스트] ← 태양광허가 API
        ↓
지번 매칭 or 좌표 반경 50m 내 매칭
        ↓
☀️ 설치완료 → 제외/흐림
⭕ 영업대상 → 표시
```

### 이격거리 판정
```python
def 이격거리_OK(지번: 좌표, 시군구):
    조례 = 조례DB.get(시군구)   # 기본값: 도로 200m / 주택 500m

    # 1. 도로 조건
    roads = 도로레이어.intersects(buffer(지번, 조례.도로_m))
    if any(r.lanes >= 2 for r in roads): return False

    # 2. 주택 군집 조건
    houses = 건축물대장.filter(주용도 IN ["단독주택","공동주택"]) \
                    .within(buffer(지번, 조례.주택_m))
    if DBSCAN(houses, eps=50m, min=5).has_cluster(): return False

    return True
```
