## 🎯 의뢰자 요청 (2026-04-17 확인)

### 요청 1: 축사/온실에 태양광 **이미 설치 여부** 표시
- **목적: 이미 설치된 곳을 피하는 용도** (영업 대상 제외)
- 대상: 50평↑ 축사/온실 (2차와 연동)

### 요청 2: 이격거리 조건 충족 부지 표시
- 왕복 2차선 도로 200m 밖 / 주택 5가구 이상 500m 밖
- 의뢰자 코멘트: "**모든 시도 조례 적용**" 원함 ("조례에 나와있으니 끌어모으면 된다")

## ✅ 검증 결론 (2026-04-17)

**전부 외부 데이터 연계 가능**. 조례 수치 추출만 LLM 반자동.
- 태양광 허가 API · 도로 SHP · 건축물대장 · 법제처 조례 API 모두 무료 공공 데이터
- 한계: 3kW 이하 자가발전 누락(허가 대상 아님 — 축사/온실은 대부분 30kW↑라 영향 적음)

## 💰 견적 초안 (의뢰자 확답 전)

- 단독 진행: **약 550만 / 10~12주** (①설치여부 + ②이격거리 옵션 B 또는 D 추천)
- 2차와 묶으면 건축물대장/VWorld 중복 제거로 100~150만 절감 가능
- 옵션 B(LLM 자동 파싱) vs D(원문 링크) — D는 유지보수 부담 0, B는 정확도↑ but 연 50~100만 유지보수

**경쟁력**: 해줌/솔라커넥트/서니로직 **아무도 이격거리 필터 없음** → 강력한 차별화
