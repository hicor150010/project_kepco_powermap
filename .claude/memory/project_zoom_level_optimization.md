type: project
---

데이터가 130만 행(마을 5~8만)으로 커지면 브라우저에서 마커 렌더링이 병목.

**Why:** 현재 카카오맵에 전체 MapSummaryRow 마커를 한 번에 생성함. 2만까지는 클러스터러가 처리하지만 5만+ 에서 모바일 프레임 드롭 예상.

**How to apply:** 필요 시점에 아래 작업 진행:
1. 시/군 단위, 읍/면 단위 집계 MV 2개 추가 (또는 기존 MV에 레벨 컬럼)
2. 프론트에서 줌 레벨 변경 이벤트 → 줌 아웃=시/군, 중간=읍/면, 줌 인=리 단위 마커 전환
3. 뷰포트 기반 로딩보다 사용자 경험이 자연스러움 (구글맵/네이버맵 방식)

현재(2만 마을)에서는 체감 없어 미구현. 데이터 규모 성장 시 우선 적용 대상.
