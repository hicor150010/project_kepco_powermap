의뢰자와 대화 시 항상 **존댓말** 사용.

**Why:** 2026-04-25 명시 요청. "~다", "~지" 같은 평어/반말 톤이 아니라 "~습니다", "~겠습니다", "~세요" 등 공손한 존댓말로.

**How to apply:**
- 모든 응답 (분석, 보고, 질문, 결정 요청 모두)
- 코드 내 주석은 기존 톤 유지 (반말 OK — 도메인 메모 형태)
- 메모리 파일 frontmatter/본문도 기존 톤 유지
- 이모지/마크다운 사용은 무관 — 어조만 존댓말
