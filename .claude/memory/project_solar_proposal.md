## 🎯 한 줄 정의

**지번 입력 → 영역 매핑 → 즉석 견적/수지분석 → 배치도/수지분석 PDF 출력**

영업 현장에서 태블릿으로 즉석 견적 + 고객 제출용 PDF 한 번에 생성하는 도구.

## 📌 의뢰자 핵심 원칙 (확정)

1. **매핑 영역 = 견적 기준** (필지 전체 ❌, 사용자가 그린 영역만 ⭕)
2. **시설 유형별 평수/단가 다름** — 4종 자동 보정 + 사용자 변경 가능
3. **단순 견적기 ❌** — 수지분석 + PDF 출력까지 풀 시스템
4. **일사량 데이터 별도 불필요** — `일평균 발전시간 4시간` 변수로 처리
5. **모든 변수 관리자 조정 가능**

## 📦 5단계 개발

### 1단계 — 지적편집도 + 필지 클릭 정보 카드 (1~2주, 100만)
- 카카오 지적편집도(`MapTypeId.USE_DISTRICT`) ON/OFF
- 필지 클릭 → 좌측 카드: **지번/주소/지목/면적/KEPCO 여유선로(3줄)**

### 2단계 — 면적 산출 + 시설별 견적 (2~3주, 170만)
- 지번 입력 시 자동 분석: 필지/건물/빈 토지
- 시설 유형 4종 자동 추천 + 사용자 변경
- 영역 선택: 옥상 / 빈 토지 / 직접 그리기
- **사용자가 그린 영역만 견적 기준**
- 자동 산출: 면적 / 패널수 / kW / 시공비

### 3단계 — 건물 자동 구분 + 패널 시각화 (2~3주, 180만)
- 필지 안 건물 1동/2동/N동 자동 구분 (봉남리 PDF 사례)
- 위성사진 위 **실제 태양광 패널 그리드** 시각화 (단순 빨간 사각형 ❌)
- 이격거리 자동 적용
- 사용자 미세 조정 가능

### 4단계 — 배치도 PDF (A3) (1~2주, 100만)
- 봉남리 배치도 PDF 양식 그대로
- 위성사진 + 패널 + 발전설비개요 + 모듈배치표 + KEPCO여유 + 회사정보

### 5단계 — 수지분석 엑셀/PDF (1~2주, 80만)
- 봉남리 수지분석 PDF 양식 그대로
- 20년 시계열 (발전량/SMP/REC/유지보수/순수익/누적/상환액)
- 3가지 시나리오: **자기자본 100% / 10년 대출 / 20년 대출**
- ROI / 손익분기점 / 20년 총수익
- 엑셀(.xlsx) + PDF 동시 다운로드

**총: 7~12주 / 630만원**

## 📦 2차 개발 (의뢰자 검토 중, 2026-04-16)

총 550만원 / 약 2개월

**스코프 확정 (2026-04-16 의뢰자 동의)**:
- 건축물대장에 정식 등록된 시설만 대상 (비닐하우스·간이 슬레이트 등 가설건축물 제외)
- 평수 기준은 **건축면적(archArea)만 사용** — 옥상 태양광 특성상 1층 바닥 면적이 곧 설치 가용 면적. 연면적(totArea)은 사용 안 함.

- **유리온실/축사 + 일반건물(평수 구간) + 여유선로 표시** — 250만 / 5주
  - 건축물대장 '동·식물 관련 시설' 필터 + KEPCO 매칭
  - 일반 건물(공장/창고 등) **건축면적** 구간 필터링 (50평↑/100평↑/150평↑/200평↑) — +50만, 기존 5주 안에 흡수
  - 시설 위 깜빡이는 원 (🔵 여유 / 🔴 부족)
- **공시지가/평수 라벨 (땅야 스타일)** — 100만 / 2~3주
  - VWorld 지적도 + 국토부 공시지가 API
- **실거래가 표시** — 100만 / 2~3주
  - 국토부 실거래가 공공 API
- **공매 표시** — 100만 / 2주
  - 한국자산관리공사(캠코) 공공 API

→ 모두 공공 API 무료, 운영비 추가 없음

**보류**: 경매 표시 — 공식 API 없음. 상용 업체(지지옥션 등) 데이터 제휴 + 월 사용료 필요. 1차/2차 효과 보고 결정.

**1차 + 2차 통합**: 1,180만 / 약 4개월

## ⚙️ 변수 기본값 (모두 관리자 조정 가능)

**견적**: 시설별 평수/시공비 (위 표)
**발전**: 일평균 발전시간 4.0h, 연 열화율 0.4%, 시스템 효율 80%
**수익**: SMP 121원/kWh, REC 72원/kWh, REC 가중치 1.5, 유지보수율 매출의 3%
**대출**: 년이율 5.5%, 거치기간 12개월

## 📐 핵심 계산 공식

```
연간 발전량(kWh) = 용량(kW) × 4.0h × 365 × 시스템효율(0.8)
N년차 발전량 = 1년차 × (1 - 0.004)^(N-1)  // 열화 0.4%
SMP수익 = 발전량 × 121원
REC수익 = 발전량 × 72원 × 1.5(가중치)
총수익 = SMP + REC
유지보수 = 총수익 × 3%
순수익 = 총수익 - 유지보수
월 상환액 = 원리금균등(대출, 5.5%, N년)
순익(대출) = 순수익 - 월상환액×12
```

PDF(봉남리 268kW, 일4h, SMP121, REC72, 가중1.5)로 검증 완료. 1년차 391,280kWh / 8,961만 매출 일치 ✅
