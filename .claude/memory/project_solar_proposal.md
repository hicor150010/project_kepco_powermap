## 서비스명 (가칭): 썬맵 자동 견적 시스템

### 벤치마크: 서니로직 (https://sunnylogic.co.kr/)

서니로직 4대 기능 탭 (스크린샷 확인됨):
1. **사업 가능성 분석** — 주소 입력 → 토지유형/이격거리/선로용량/제한지역 한 번에 확인
2. **매핑 시뮬레이션** — 건물 위에 패널 배치 시각화, AI 기반 방위각/경사각 최적화
3. **자동 제안서 출력** — 연별·월별 예상 발전량 차트 + 세부 수익안 표
4. **신규 프로젝트 알림** / 프로젝트 매칭 — 시공사↔발전사업주 매칭 플랫폼

**우리 차별화**: 서니로직은 선로용량 확인 기능이 없거나 별도. 우리는 **KEPCO 3줄 마커(변전소/주변압기/배전선로 여유) + 견적을 동시에** 보여줌. "여기 선로 남았고, 이 평수면 얼마입니다"를 한 화면에 끝냄.

**Why:** 의뢰자 비전 — "태블릿으로 고객 집 지붕을 슥슥 그어서 선로도 넉넉하고 1.2억 투자하면 월 얼마 나옵니다 바로 보여주는" 영업 도구. 데이터 기반 신뢰 = 계약률↑

## 기능 1: 자동 견적 시스템 (최우선)

### 모드 A: 지도 매핑 (자동)

1. **스카이뷰에서 건물 지붕 클릭/드로잉**
   - 다각형(폴리곤) 꼭짓점 찍어서 지붕 영역 지정
   - 위경도 좌표 기반 **면적(㎡) → 평수** 자동 계산 (1평 ≈ 3.3058㎡)

2. **패널 배치 시각화** (서니로직 핵심 UX)
   - 그린 영역 안에 **550W~600W 패널 이미지가 자동 타일링**
   - 이격거리(그림자 간격) 자동 적용
   - 면적 그리는 동안 실시간으로 패널 개수/kW 갱신

3. **자동 산출 결과**
   - 설치 가능 면적(평)
   - 패널 수량
   - 설치 용량(kW) — 예: 150~180평 ≈ 100kW
   - **시설 견적** — 기본값: **100kW당 1.2억** (= kW당 120만원)
     - 이 단가는 **설정값(변수)**으로 빼서 사용자/관리자가 조정 가능

### 모드 B: 수동 입력

- 매핑 없이 **평수만 직접 입력** → 견적 즉시 산출
- 현장 방문 시 대략적 면적만 알면 바로 견적 뽑기 가능
- 평수 입력 → kW 환산 → 견적 자동

### 산출 공식

```
총 견적 = 설치 가능 용량(kW) × 단가(기본 120만원/kW)
```

**설정 가능한 변수 (관리자/사용자 조정)**
| 변수 | 기본값 | 설명 |
|---|---|---|
| kW당 단가 | 120만원 | 시세에 따라 조정 (100~140 범위) |
| 패널 와트 | 550~600W | 사용 패널 사양 |
| 이격거리 계수 | 표준값 | 지붕 경사/방향에 따라 조정 |

## 기능 2: 수익분석 (자동 + 수동)

### 자동 모드
- 견적 산출 후 **자동으로 수익 시뮬레이션 팝업**
- 기본 시나리오 2개 즉시 표시:

| 시나리오 | 자기자본 비율 | 상환기간 | 금리 |
|---|---|---|---|
| A | 100% 대출 | 10년 | 5% |
| B | 100% 대출 | 20년 | 5% |

### 수동 모드
- 사용자가 직접 조정 가능한 슬라이더/입력:
  - 자기자본 비율 (0~100%)
  - 대출 상환기간 (5/10/15/20년)
  - 금리 (1~10%)
  - kW당 단가 조정

### 출력 항목
- 월 예상 발전량 (kWh)
- 연간 매출 (SMP + REC)
- 월 대출 상환액
- **월 순수익** (매출 - 상환액)
- 투자 회수 기간
- 20년 누적 수익
- (서니로직 벤치마크) **연별·월별 예상 발전량 차트** + 세부 수익안 표

### 산출 공식

```
연간 발전량(kWh) = 용량(kW) × 일 발전시간(약 3.3h) × 365 × 효율
연간 매출 = 발전량 × (SMP + REC 단가)
월 상환액 = 원리금균등상환(대출금, 금리, 기간)
월 순수익 = 월 매출 - 월 상환액
회수기간 = 총 투자 / 연간 순수익
```

## 기능 3: 태양 각도 최적 효율 이미지화

서니로직 핵심 기능: **"AI 기반 최적의 방위각과 경사각"** 시각화

### 구현 내용
- 해당 위치(위도) 기반 **최적 경사각(틸트)** 계산 — 한국 평균 30~33°
- **방위각** — 정남(180°) 기준, 동서 편차 시 발전량 감소율 표시
- **3D 이미지화**: 패널이 지붕 위에 놓인 모습 + 태양 궤적 시각화
  - 서니로직: 위성사진 + 패널 3D 렌더 + 최적 방위 165° / 최적 효율 21° 같은 수치 오버레이
- 월별/계절별 태양 고도 변화 시뮬레이션 (선택)
- 라이브러리 후보: `suncalc` (JS, 태양 고도/방위각), Three.js (3D 렌더), 또는 PVLib (서버)

## 기능 4: 사업 가능성 원클릭 분석 (서니로직 벤치마크)

서니로직 "완료 포인트 분석" 화면:
- 토지유형 → 가능 ✅
- 이격거리 확인 → 가능 ✅
- **선로용량확인** → 가능 ✅ ← **우리가 이미 가진 핵심 데이터**
- 제한 지역 → 확인 필요 ⚠️

### 우리 구현
주소 입력 또는 지도 클릭 → 한 화면에:
1. ✅ 선로 여유 (3줄 마커 — 변전소/주변압기/배전선로)
2. ✅ 토지유형/지목 (공공데이터 API)
3. ✅ 개발행위 규제 (eum.go.kr)
4. ✅ 이격거리 자동 계산 (건물 폴리곤 기반)

## 차별화 포인트 (vs 서니로직)

- **여유용량 실시간 연동**: 3줄 마커 + 견적 = 한 화면에서 "설치 가능 여부 + 비용"
- **변화추적**: "이 지역 최근 여유 생겼으니 지금이 기회" 식 영업 가능
- **타겟 오버레이**: 유리온실/축사/대형창고 자동 필터 → 영업 리스트
- **특허 청구항 포인트**: "지도 데이터 상의 다각형 영역 지정에 따른 배전 여유 용량 기반 발전 설비 최적 규모 산출 및 공사비 산출 알고리즘"
